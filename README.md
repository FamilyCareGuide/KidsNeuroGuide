# Kid Neuro Guide Website

A ready-to-publish static website based on the PDF guide: "What to Expect at a Pediatric Neurologist Appointment."

## Publish on GitHub Pages
1. Create a free GitHub account.
2. Create a new public repository named `kid-neuro-guide`.
3. Upload all files from this folder.
4. Go to Settings -> Pages.
5. Under "Build and deployment," choose "Deploy from a branch."
6. Choose `main` and `/root`, then Save.
7. Your site will appear at `https://YOUR-USERNAME.github.io/kid-neuro-guide/`.

## Before publishing
- Replace `YOUR-DOMAIN-HERE` in `robots.txt` and `sitemap.xml` with your final URL.
- Replace any resource links if your teacher wants different sources.
- This site is educational only and not medical advice.
